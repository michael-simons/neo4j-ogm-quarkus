== Installation

If you want to use this extension, you need to add the `io.quarkiverse.neo4j:quarkus-neo4j-ogm` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.neo4j</groupId>
    <artifactId>quarkus-neo4j-ogm</artifactId>
</dependency>
----
